官网页面结构说明

本文档用于将前台官网从“栏目级规划”推进到“页面级规划”。 每个页面明确页面目标、模块构成、内容来源与新补充内容，便于后续原型设计、文案整理、视觉设计和前端开发直接衔接。

11个页面 覆盖本期前台范围
转化优先 首页/方案/联系为重点
内容重构 不是机械迁移旧站
中英独立 避免中英混排

前台核心页面

首页
优先级 P0 / 核心转化页

页面目标

  • 一句话讲清是谁、做什么、服务谁
  • 集中呈现品牌、业务、工厂、品质、案例
  • 将访客导向产品页、方案页、联系我们

页面模块

  • 首屏 Banner 与品牌定位
  • 公司简介与核心数据
  • 两大业务方向入口
  • 核心优势、案例预览、工厂预览、新闻预览、联系表单

内容来源

  • 旧站简介、服务、设备、品质内容可提炼
  • 需新增品牌主张、CTA、案例图、核心数据
  • 建议首页底部直接放询盘表单
关于我们
优先级 P0 / 信任页

页面目标

  • 建立企业可信度
  • 说明的发展背景与技术积累
  • 承接资质、团队、工厂实力表达

页面模块

  • 公司简介
  • 发展历程
  • 荣誉资质
  • 团队 / 企业文化 / 合作客户

内容来源

  • 旧站 company.html 可作为基础文案
  • 需补发展历程、证书图、团队素材
  • 建议强化“2004年成立”和高新技术定位
产品中心
优先级 P0 / 业务核心页

页面目标

  • 把工艺能力结构化为可浏览内容
  • 让客户快速找到相关膜层或处理能力
  • 形成标准化产品详情模板

页面模块

  • 分类页：装饰膜、功能膜、光学膜、离子硬化
  • 列表页：卡片、筛选、封面图、应用标签
  • 详情页：材质、颜色、性能、应用、咨询按钮

内容来源

  • 旧站 service.html 内容丰富，可拆解复用
  • 需补图片、标准字段、详情页结构
  • 不建议沿用旧站“长篇技术说明”排版
解决方案
优先级 P0 / 核心转化页

页面目标

  • 按行业场景组织内容，而不是按内部工艺分类
  • 让客户从自身行业痛点切入理解价值
  • 承接方案咨询与定制需求

页面模块

  • 方案列表：智能穿戴、消费电子、传统五金、汽车、医疗
  • 方案详情：行业痛点、推荐工艺、典型部件、案例、CTA
  • 相关产品与相关案例联动

内容来源

  • 旧站服务页和简介页有行业信息可提炼
  • 需按行业重新组织文案
  • 建议将“镜头圈、按键、Logo标识”等重点突出
成功案例
优先级 P1 / 信任增强页

页面目标

  • 证明真实交付能力
  • 增强客户对品质、稳定性、量产能力的信任
  • 支撑销售沟通与品牌背书

页面模块

  • 案例列表与行业筛选
  • 案例详情：背景、需求、工艺、结果、图片
  • 相关推荐与咨询入口

内容来源

  • 旧站几乎没有成型案例内容
  • 需要业务侧补项目素材和可公开范围
  • 若品牌限制强，可用匿名行业案例
工厂展示
优先级 P1 / 制造信任页

页面目标

  • 突出制造能力、自动化和品质管控
  • 把旧站设备页和品质页内容可视化表达
  • 提升制造业官网最关键的“可信度”

页面模块

  • 工厂环境
  • 核心设备
  • 自动化产线
  • 实验室与检测能力、生产流程、产能表达

内容来源

  • 旧站 equipment.html 与 quality.html 可提炼
  • 需要大量新图片和可能的视频素材
  • 建议避免纯文字堆叠
新闻资讯
优先级 P1 / 运营与SEO页

页面目标

  • 支撑内容更新与搜索曝光
  • 展示公司动态、行业观点和公告信息
  • 维持官网活跃度

页面模块

  • 新闻分类：公司新闻、行业资讯、公告通知
  • 列表页、详情页
  • 相关文章与分享入口

内容来源

  • 旧站仅有公告页，可迁移人权政策等信息
  • 新闻体系基本需新建
  • 建议预留图文富文本能力
技术支持
优先级 P1 / 能力沉淀页

页面目标

  • 把旧站工艺与品质说明沉淀为结构化知识
  • 方便客户理解工艺、标准和适配范围
  • 支撑业务沟通与专业度表达

页面模块

  • 工艺介绍
  • 检测标准
  • FAQ
  • 下载中心

内容来源

  • 旧站 service.html、quality.html 内容可复用
  • 需重新编排成问答和文章结构
  • 后续可持续扩展为 SEO 内容库
视频中心
优先级 P2 / 补强页

页面目标

  • 提升工厂、工艺和企业展示的直观感受
  • 支持对外展示宣传片与产线视频
  • 增强信任感

页面模块

  • 视频分类
  • 视频列表
  • 视频详情或弹窗播放
  • 封面、简介、标签

内容来源

  • 旧站无成熟视频内容
  • 需要新补企业宣传片或产线短视频
  • 可先做结构，内容后续补齐
联系我们
优先级 P0 / 最关键转化页

页面目标

  • 承接所有咨询和合作意向
  • 降低客户联系成本
  • 形成后台可追踪线索

页面模块

  • 地址、电话、邮箱、地图
  • 在线询盘表单
  • 海外客户联系方式说明
  • 提交成功反馈

内容来源

  • 旧站 contact.html 可直接迁移基础联系方式
  • 需新增在线表单和后台管理链路
  • 建议全站固定悬浮 CTA 导向本页
招贤纳士
优先级 P1 / 招聘转化页

页面目标

  • 承接招聘信息发布与简历收集
  • 增强雇主品牌形象
  • 配合后台招聘管理使用

页面模块

  • 招聘介绍
  • 岗位列表
  • 岗位详情
  • 简历投递表单

内容来源

  • 旧站无该页面内容
  • 需新增岗位和 HR 文案
  • 需支持附件上传与后台查看
页面设计原则：先建立品牌与信任，再组织业务与能力，最后在核心节点承接转化。 对旧站内容的处理方式应当是“重组 + 精简 + 结构化”，而不是直接复制长篇技术文案。
