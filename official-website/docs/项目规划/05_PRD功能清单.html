官网 PRD 功能清单

本文档用于将前台官网、后台管理和公共能力整理为可执行的产品功能清单。 文档重点强调范围边界、转化链路、内容可维护性和实施优先级，为后续数据库设计、接口规划和开发任务拆解提供基础。

前台功能清单

功能模块 功能说明 关键点 优先级 备注
首页展示 展示品牌定位、核心优势、业务方向、案例预览、工厂实力、新闻预览和联系入口。 首屏表达清晰、强 CTA、支持中英文。 P0 核心入口页。
关于我们 展示公司简介、发展历程、荣誉资质、企业文化、合作客户等内容。 建立企业可信度。 P0 承接品牌信任。
产品中心 支持按分类展示产品或工艺能力，包含列表页和详情页。 标准化字段、支持图片、应用场景、咨询按钮。 P0 旧站内容需重构。
解决方案 按行业展示场景方案，如智能穿戴、消费电子、传统五金、汽车、医疗等。 行业痛点、推荐工艺、案例联动、表单承接。 P0 核心转化页。
成功案例 案例列表、案例详情、行业筛选、图文介绍。 真实案例、可公开范围需确认。 P1 增强信任。
工厂展示 展示工厂环境、设备、自动化产线、品质实验室、生产流程。 图片和视频优先于大段文字。 P1 制造业官网重点页。
新闻资讯 支持公司新闻、行业资讯、公告通知列表和详情展示。 分类管理、发布时间、富文本详情。 P1 支撑 SEO 与更新。
技术支持 展示工艺介绍、检测标准、FAQ、下载资料等内容。 结构化知识沉淀。 P1 可持续扩展。
视频中心 展示企业宣传片、工艺视频、产线视频和案例视频。 视频封面、分类、播放方式。 P2 可后补素材。
联系我们 展示联系方式、地图、在线询盘表单。 全站转化承接、提交成功反馈。 P0 最重要转化页。
招贤纳士 展示岗位列表、岗位详情、简历投递表单。 附件上传、岗位关联。 P1 招聘转化页。

后台功能清单

功能模块 功能说明 关键点 优先级 备注
登录与权限 后台账号登录、退出、角色与权限控制。 管理员、编辑、招聘/销售可扩展。 P0 所有后台基础。
站点配置 维护站点标题、联系方式、页脚、备案、社媒、语言配置等。 支持统一配置，避免写死。 P0 全局配置项。
首页配置 维护首页 Banner、核心数据、业务入口、案例预览、联系信息等。 模块化配置、排序控制。 P0 便于运营更新。
关于我们管理 维护公司介绍、发展历程、资质证书、企业文化等。 支持多图、富文本、中英文。 P0 品牌内容维护。
产品管理 维护产品分类、产品详情、材质、颜色、性能、应用场景。 分类与详情分离、支持上下架。 P0 建议字段化设计。
解决方案管理 维护行业方案、场景描述、推荐工艺、关联产品与案例。 支持关联内容。 P0 转化价值高。
案例管理 维护案例列表、行业分类、封面图、详情和排序。 支持置顶、行业筛选。 P1 可扩展匿名案例。
工厂展示管理 维护工厂图片、设备介绍、实验室和产线内容。 图文组合展示。 P1 偏内容模块。
新闻管理 维护新闻分类、列表、详情、发布时间、上下架。 富文本、封面图、SEO 字段。 P1 支撑 SEO。
技术支持管理 维护技术文章、FAQ、下载资料。 结构化问答和附件支持。 P1 后续 SEO 积累点。
视频管理 维护视频分类、封面、简介、播放链接或文件地址。 封面与链接统一管理。 P2 可首期简化。
招聘管理 维护岗位分类、岗位详情、上下架状态。 岗位与简历关联。 P1 HR 可用。
简历管理 查看投递简历、下载附件、标记状态、记录备注。 附件安全与状态维护。 P1 招聘闭环。
询盘管理 查看客户询盘、状态维护、备注跟进、导出预留。 支持附件和来源字段。 P0 销售线索闭环。
文件管理 统一管理图片、视频、PDF、附件资源。 文件分类、路径、引用追踪。 P0 全局能力。
SEO 管理 维护页面 title、description、keywords、URL 别名等。 页面级可配置。 P1 建议统一封装。

公共能力与业务规则

类别 说明 要求 备注
多语言 中文、英文内容独立维护，前台支持切换。 P0 避免中英混排。
响应式 适配桌面端、平板、移动端。 P0 制造业客户常用手机查看。
表单提交 询盘和简历表单需有校验、成功提示、后台入库。 P0 核心转化链路。
上传能力 支持图片、PDF、简历附件等上传。 P0 需限制类型和大小。
内容状态 支持草稿、发布、下架或启用、禁用。 P1 便于运营审核。
排序能力 Banner、产品、案例、新闻等支持排序。 P1 首页运营需要。
安全性 后台接口鉴权、输入校验、文件安全控制。 P0 基础要求。
SEO 页面标题、描述、关键词、语义化结构可控。 P1 需和前端渲染方案联动。
这份功能清单已经与后续的数据库、接口、后台菜单、测试和上线文档形成闭环。 审核时建议重点关注功能边界、优先级划分和后台是否真正服务于官网运营。
